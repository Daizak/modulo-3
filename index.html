1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. CSS permite separar el contenido de la presentación, lo que facilita el diseño y la adaptación de las páginas web a diferentes dispositivos.

2. Funcionamiento Básico de CSS.

El funcionamiento básico de CSS se basa en la aplicación de estilos a los elementos HTML. Esto se logra mediante la selección de elementos y la asignación de propiedades y valores que determinan cómo se verán en la pantalla. CSS permite controlar aspectos como colores, fuentes, márgenes, y disposición de los elementos.

4. Definición de una regla CSS.

Una regla CSS es una declaración que define cómo se deben presentar ciertos elementos en un documento HTML. Consiste en un selector y un bloque de declaración.

5. Elemento de una regla CSS.

  • Regla

    • Conjunto de instrucciones que aplican estilos a los elementos seleccionados.

  • Selector

    • : Parte de la regla que especifica a qué elementos se aplicará el estilo

  • Bloque de declaración

    • Contiene una o más declaraciones de estilo, encerradas entre llaves {}.

  • Propiedad

    • Aspecto del estilo que se desea modificar (por ejemplo, color, font-size).

  • Valor

    • Especifica el valor que se asigna a la propiedad (por ejemplo, red, 16px).

6. Medios CSS.

Los medios CSS permiten aplicar estilos específicos según el tipo de dispositivo o las características del entorno de visualización.

7. Comentarios CSS.

Los comentarios CSS se utilizan para incluir notas o explicaciones dentro del código sin que afecten la presentación. Se escriben encerrando el texto entre /* y */.
